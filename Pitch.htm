Pitch title: Hybrid Tower Defence Game Thing

Main Concept

Think tower defence meets moba characters. Waves of enemies will come from one end of a path and make their way to the other end.

The player will have a hero with basic attacks and a few skills along with the ability to place towers. These towers will fire on approaching enemies or deploy troops of their own to block and fight enemies.

I'd like to have upgrade tree for each tower that can be bought over the course of each level as well as hero upgrades which persist between levels.

I'm planning to use a single quadtree for pathfinding as well as tower placement (collision, restricted areas) by subdividing the game map between path/grass/impassable regions.

Long Version

Experience Goals

The player will be required to think strategically with the tower defence elements but will also have to master their hero character.

Platforms

Windows. Much surprise. Many shock. Wow. 

The game will require a controller or a mouse and keyboard.
